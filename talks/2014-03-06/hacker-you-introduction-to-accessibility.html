What Can I Do?

  • Visibility

    Bad

    • Using z-index to hide objects
    • Using off screen space to hide objects
    • Using opacity: 0; to hide objects

  • Good

    • Using hidden attribute (if supported)
    • Using display: none; style
    • Using visibility: hidden; style

What About More Complicated Web Application?

  • ARIA (Accessible Rich Internet Applications Suite)

    Defines a way to make Web Content and Web Applications more accessible to people with disabilities.

    http://www.w3.org/WAI/intro/aria

  • What is ARIA?

    ARIA is an extension of the HTML.

    ARIA does not require any knowledge of other technologies.

    You can start using ARIA by adding attributes to your HTML elements.

Aria Roles

Indicate the type of the element. Provides the semantics.

  • Examples of Roles

    • Widget Roles: button, alert, dialog, tablist, tabpanel, tab...
    • Document Structure Roles: article, math, presentation, toolbar...
    • Landmark Roles: main, search, complementary...

Aria States and Properties

Provide specific Information about an object.

  • Examples of States and Properties

    • Global States and Properties: aria-describedby, aria-disabled, aria-controls, aria-labelledby...
    • Widget Attributes: aria-checked, aria-haspopup, aria-autocomplete...
    • Relationship Attributes: aria-posinset, aria-setsize...
